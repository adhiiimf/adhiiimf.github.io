De-Lansia App

Android App Development

Android Monitoring App to detect eldery condition like walking, falling, and sit down. And then how to detect? we use AI to train the device and the monitor via Android App. Available on Play Store.

Research and project under Mrs. Husneni Mukhtar, S.Si, M.T., Ph.D and Mr. Dr. Eng. Willy Anugrah Cahyadi, S.T., M.T..

How It Works

Before I Tell you about How It Works, I think will be better if I describe the background of my role in this project.

First, this projects is collaboration research between FTE Telkom University and PT. Dycodex Teknologi Nusantara. So my role in here is as a Software Developer to handle and build Back-End (DB Archicecture, DBMS, and etc.) and Front-End (UI UX Design, Android App, etc.).

Don't Worry, i'll describe overall of what I've done in this project. So First, I develop Back-End with PostgreSQL as DBMS (actually only psql because desktop version made my PC slow, I'm Sorry..) and PlatformaticJS for create RESTful API. This is example (basic of what i've done in first time). So next image is one of documentation when i develop DB. (Sorry i can't give you more documentions about backend because they're private things)
Next, I Develop Android App with Flutter (from dart language). So I've implemented mockup from FIGMA that I've designed to Android App. But for implemented that things, I'm Not Alone, I've friend just call Iki, he as Front-End Develop to develop UI, and after he developed that all I'll review and if it's OK -> I'll merge with Back-End Function.

So, How it works? De-Lansia has a chip on wearable device that have AI Models to predicted and classified Eldery condition. And from the chip, the result of classify will be send to cloud database on azure. And Android App will do a Pull Data from that cloud DB. So what next? The App will be show the data depends of latest data in DB. If latest data is 'Falling', The App will be notifying the user to tell that the Eldery is falling, and location will be updated.
